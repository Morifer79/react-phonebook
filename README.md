  A simple phonebook app with the ability to edit contacts (add or delete) and filter by current base.  
  
※ @emotion/react and @emotion/styled: Styling library for React components.  
※ prop-types: to define the expected data types.  
※ formik and yup: for easy handling and validation of the submission form.    
※ nanoid: for random id generation.    

## 𝕀𝕟𝕤𝕥𝕒𝕝𝕝𝕒𝕥𝕚𝕠𝕟 ᎓  

To get started with this project, follow the installation instructions below.

1. Clone the repository:
   ```bash
   git clone https://github.com/Morifer79/react-phonebook.git
   cd react-phonebook-app
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
4. Open to view it in the browser:
